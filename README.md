# neo-banker/.github

This is the [special GitHub org repo](https://docs.github.com/en/organizations/collaborating-with-groups-in-organizations/customizing-your-organizations-profile) for [neo-banker](https://github.com/neo-banker).

## What's here

- `profile/README.md` — public-facing org profile (rendered at https://github.com/neo-banker)

## Why it exists

To give visitors (humans + AI agents) a single entry point that:
1. Explains what neo-banker is (platforms / products)
2. Lists all repos with one-line purpose
3. Tells AI agents which `AGENT_HANDOFF.md` to read in each repo

## Not here (intentionally)

- No issue templates / PR templates (each repo has its own)
- No workflows (each repo has its own CI)
- No CODEOWNERS (per-repo)
